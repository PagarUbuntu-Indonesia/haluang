### <i class="fa fa-code"></i> CASE ... WHEN ... {:.titleMe }

Sintaks 1:
```sql
CASE 
  WHEN condition1 THEN result1
  WHEN condition2 THEN result2
  ...
  ELSE resultn
END
```

Sintaks 2:
```sql
CASE expression
  WHEN value1 THEN result1
  WHEN value2 THEN result2
  ...
  ELSE restuln 
END
```

Contoh:
```sql
sqlite> SELECT nama,
   ...> CASE
   ...>   WHEN id_kategori IS NULL THEN 'Ga ada kategori'
   ...>   ELSE id_kategori
   ...> END
   ...> AS KategoriMenu
   ...> FROM menu;
```
